# final-oop-2026-03-11
Contiene la evaluación final correspondiente a la materia POO (programación orientada a objetos) que fue tomada el 11 de marzo del 2026. Posee tanto la versión para el **alumno** como la versión **solución**. Con test automatizados que inician en rojo y es el propio alumno el encargado de pasarlos a verde a medida que soluciona los requerimientos.
## Los principales directorios y archivos son: 
* ### “Final_2026 (Para el alumno)”: 
Contiene la versión que el docente entregara al alumno para que resuelva. Incluye los tests automatizados.
* ### “Final_2026 (Solucion)”: 
Contiene la versión con el código solución, que el docente entregara al alumno luego de la finalización del examen. Incluye los tests automatizados, que ahora darán todos verdes.
* ### “CREATE TABLE [Cuenta].sql”: 
Este archivo contiene el script de **T-SQL** responsable de la creación de la base de datos, que el alumno debe ejecutar debido a que la aplicación depende de la misma.
